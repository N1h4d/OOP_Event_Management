# User Guide – Event Management System

## 1. Requirements

- Python 3 installed
- No external libraries required (SQLite is built-in)

---

## 2. How to Start the Application

```bash
cd OOP_Event_Management
python3 -m src.main
```

## 3. Unit Testing

Unit tests are implemented using Python's built-in unittest framework:
✅ Model tests (test_models.py)
✅ Repository tests with in-memory SQLite (test_repositories.py)

python3 -m unittest
